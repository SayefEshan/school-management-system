## Helpers
### Every Files in this directory will be autoloaded by Laravel.

Notes:
- Any helper functions that are used throughout the application should be placed here.
- Helper functions should be stateless and should not depend on any external state.
- Helper functions should be pure functions, meaning they should not have any side effects.
- Helper functions should be unit tested.
- Helper functions should be documented.
- Helper functions should be namespaced.
- Helper functions should be used in the application.
